# Changelog

All notable changes to this project will be documented in this file.

## 0.0.1 - 2026-04-09

Initial Module 1 release.

- FastAPI backend with auth-protected thread and message routes.
- React and Vite frontend with Supabase auth and protected chat UI.
- Supabase schema and row-level security for threads and messages.
- Azure OpenAI streaming chat responses over SSE.
- LangSmith tracing for backend response generation.
- End-to-end validation covering auth flow, chat flow, and backend health.
